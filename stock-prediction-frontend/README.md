# Stock Price Prediction Frontend

React frontend for the Stock Price Prediction application using LSTM Neural Networks.

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Backend Setup

Make sure the Flask backend is running on `http://localhost:5000` before using this frontend.

Start the backend:
```bash
python app.py
```

## Features

- Input form for stock ticker and date range
- Real-time stock price predictions using LSTM
- Interactive charts showing actual vs predicted prices
- RMSE metrics display
